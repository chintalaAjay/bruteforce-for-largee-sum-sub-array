Maximum Subarray Sum using Brute Force (Java)

This project finds the maximum sum of a contiguous subarray using the Brute Force approach in Java.

🚀 Problem Statement

Given an integer array containing positive and negative numbers, find the contiguous subarray with the largest sum.

Example

Input:

{2,4,-2,9,8,-2,-1,6}

Output:

24
💡 Approach Used

This solution uses:

Three nested loops
Generates all possible subarrays
Calculates the sum for each subarray
Tracks the maximum sum found
🧠 Algorithm
Traverse the array using index i
Generate subarrays from j to i
Calculate subarray sum using another loop
Compare and update maximum sum
⏱️ Complexity
Time Complexity: O(n³)
Space Complexity: O(1)
